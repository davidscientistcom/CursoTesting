1) API de pagos: evitar regresiones en reglas de negocio

Aporta: pytest + hypothesis. Evita romper reglas de importes, comisiones y límites por edge cases.

python · pytest + hypothesis
from decimal import Decimal
from hypothesis import given, strategies as st


def calc_fee(amount: Decimal) -> Decimal:
    if amount <= Decimal('0'):
        raise ValueError('amount must be positive')
    # 1.5% con tope de 20
    fee = amount * Decimal('0.015')
    return min(fee, Decimal('20'))


def test_known_case():
    assert calc_fee(Decimal('100')) == Decimal('1.5')


@given(st.decimals(min_value='0.01', max_value='50000', places=2))
def test_fee_never_exceeds_cap(amount):
    fee = calc_fee(amount)
    assert fee <= Decimal('20')
    assert fee > Decimal('0')

2) ETL de datos: validar calidad y contratos de esquema

Aporta: pytest + pandas + validaciones. Detecta columnas rotas antes de cargar al warehouse.

python · test de contrato de datos
import pandas as pd

REQUIRED = {'order_id', 'user_id', 'amount', 'created_at'}


def validate_orders(df: pd.DataFrame):
    missing = REQUIRED - set(df.columns)
    if missing:
        raise ValueError(f'Missing columns: {missing}')
    if (df['amount'] <= 0).any():
        raise ValueError('amount must be > 0')


def test_orders_contract():
    df = pd.DataFrame([
        {'order_id': 1, 'user_id': 9, 'amount': 30.5, 'created_at': '2026-03-10'}
    ])
    validate_orders(df)  # no exception

3) Microservicio asíncrono: prevenir timeouts silenciosos

Aporta: pytest-asyncio + pytest-mock. Permite probar reintentos y fallback en `async`.

python · test async con mock
import asyncio
import pytest


async def fetch_with_retry(client, retries=2):
    for i in range(retries + 1):
        try:
            return await client.fetch()
        except TimeoutError:
            if i == retries:
                raise
            await asyncio.sleep(0)


@pytest.mark.asyncio
async def test_retry_success(mocker):
    client = mocker.Mock()
    client.fetch = mocker.AsyncMock(side_effect=[TimeoutError(), {'ok': True}])
    res = await fetch_with_retry(client)
    assert res == {'ok': True}
    assert client.fetch.call_count == 2
